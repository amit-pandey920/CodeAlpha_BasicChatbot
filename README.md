# Basic Chatbot

## Description
This is a simple rule-based chatbot developed using Python.

## Features
- Responds to greetings like "hello"
- Answers "how are you"
- Tells its name
- Provides help commands
- Exits when user types "bye"

## Technologies Used
- Python
- if-elif statements
- Functions
- Loops
- Input/Output

## How to Run
1. Install Python.
2. Download the project files.
3. Open terminal in the project folder.
4. Run:

```bash
python chatbot.py
