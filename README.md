# Basic Chatbot

## Description

This is a simple rule-based chatbot developed using Python as part of the CodeAlpha Internship Program.

The chatbot interacts with users by responding to predefined inputs such as greetings, questions, and exit commands.

## Features

* Responds to greetings like "hello"
* Answers "how are you"
* Tells its name
* Provides help commands
* Exits when the user types "bye"
* Handles unknown inputs gracefully

## Technologies Used

* Python
* Functions
* if-elif statements
* Loops
* Input/Output Operations

## How to Run

1. Install Python on your computer.
2. Download or clone this repository.
3. Open the terminal or command prompt in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

## Sample Commands

* hello
* how are you
* what is your name
* help
* bye
